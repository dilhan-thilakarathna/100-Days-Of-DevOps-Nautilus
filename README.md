## 📖 Project Overview
Welcome to my DevOps engineering portfolio. I am documenting my 100-day experiential learning journey through KodeKloud, operating as a Systems Administrator and DevOps Engineer for **Project Nautilus Marine**, the naval subdivision of XFusion Corps Industry. 

This repository serves as a technical log of my progression from foundational Linux administration to deploying automated, scalable cloud infrastructure using Infrastructure as Code (IaC) and CI/CD pipelines.

**Author:** Dilhan Thilakarathna
